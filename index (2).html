📡 GA4 + Microsoft Clarity Setup

Tracking architecture that connects every ad dollar to real business outcomes — not vanity metrics. Set this up before spending a single rupee on ads.

📊
GA4 — Events to Configure

  • audit_form_submit — Primary conversion. Weight: 10 points.
  • calendar_booking_confirmed — Highest value conversion.
  • thank_you_page_view — Fallback confirmation event.
  • cta_button_click — Tracks all CTA button interactions by button label.
  • scroll_depth_75 — Engagement quality for landing page.
  • video_play + video_complete — For any video on landing page.
  • phone_click — Tap-to-call tracking on mobile.
  • whatsapp_click — WhatsApp CTA engagement.
  • time_on_page_120s — Engagement signal for retargeting.
  • exit_intent_popup_shown — If using exit popup.

🧠
Microsoft Clarity — Setup Goals

  • Enable session recordings — watch how founders interact with landing page.
  • Enable heatmaps — identify where attention drops.
  • Set up Smart Events: Form interaction, CTA click, Scroll 75%.
  • Filter by traffic source — compare Google vs Meta vs LinkedIn behavior.
  • Enable Rage Click filter — identifies UX friction in form/CTA areas.
  • Create Segments: Bounced visitors, Form abandoners, Mobile vs Desktop.
  • Connect Clarity with GA4 — enables UserID matching for full journey view.
  • Set up Funnel in Clarity: Hero → Pain Section → CTA → Thank You.
  • Tag recordings by UTM source for ad-specific UX analysis.
  • Weekly review: Watch top 5% longest sessions for insight mining.

🔗
GA4 — Conversion Goals Setup

  • Mark audit_form_submit as Primary Conversion in GA4 Admin.
  • Mark calendar_booking_confirmed as Primary Conversion.
  • Assign monetary value to each conversion (estimate LTV ÷ close rate).
  • Import GA4 conversions into Google Ads for Smart Bidding.
  • Create Audiences: All Visitors, CTA Clickers, Form Starters (no submit).
  • Enable Enhanced Measurement (outbound clicks, file downloads, scrolls).
  • Connect GA4 to Google Ads, Meta (via Conversions API), LinkedIn Insight Tag.
  • Set up channel groupings: Paid Search, Paid Social, Organic, Referral.
  • Create custom report: Leads by Source × Landing Page × Ad Campaign.
  • Set up weekly automated email report to founder dashboard.

📋
UTM Tagging Convention

  • utm_source: google / meta / linkedin
  • utm_medium: cpc / paid-social / message-ad
  • utm_campaign: tofu-pain-hook / mofu-case-study / bofu-retarget
  • utm_content: headline-variant-1 / video-30s / doc-ad
  • utm_term: {keyword} for Google, audience name for Meta/LinkedIn
  • Always use lowercase, hyphens not underscores.
  • Create a UTM builder spreadsheet for team consistency.
  • Never use auto-tagging AND manual UTMs together on same URL.
  • Test all URLs with Google Tag Assistant before going live.
  • Monthly audit: check for broken UTMs in GA4 Source/Medium report.
